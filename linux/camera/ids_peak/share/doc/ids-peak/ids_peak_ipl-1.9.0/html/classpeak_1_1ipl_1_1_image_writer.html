Detailed Description

Writes an image to a file.

Supported Formats are currently: JPEG, PNG and BMP and a simple RAW format

const Image &  imageToSave

inlinestatic

Writes the specified image to the filesystem as BMP image.

Not all image formats can be written to a BMP file. Currently supported for: Mono8, Mono10, Mono12, RGB8, BGR8, RGBa8, BGRa8 Written as Mono: BayerGR8, BayerRG8, BayerGB8, BayerBG8, BayerGR10, BayerRG10, BayerGB10, BayerBG10, BayerGR12, BayerRG12, BayerGB12, BayerBG12 For all other formats an exception is thrown.

Parameters
[in]filePathThe path of the file to store the image to as an UTF-8 encoded string.
[in]imageToSaveThe image to save.
Exceptions

ExceptionAn internal error has occurred
