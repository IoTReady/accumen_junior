peak::ipl::Image peak::ipl::ImageReader::Read ( const std::string &  filePath,
const peak::ipl::PixelFormat &  pixelFormat 
)
inlinestatic

Returns a new created image which is read from the file with the given file path.

The Pixel format to use is specified manually here. The function tries to interpret the file with the given format. If this isn't possible, an exception is thrown. Explicit conversion of image formats must be done manually.

Parameters
[in]filePathThe path to the file to read as an UTF-8 encoded string.
[in]pixelFormatThe Pixel format the retrieved image should have afterwards.
Exceptions
ImageFormatInterpretationExceptionCan't interpret this file with the given pixel format
ImageFormatNotSupportedExceptionAn image format of this file isn't supported
IOExceptionErrors during file access e.g. no permissions on this file
InvalidArgumentExceptionArguments passed are invalid
ExceptionAn internal error has occurred
