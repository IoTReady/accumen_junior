◆ Exit()

void Exit ( )
inline

Exit the peak_afl auto feature library.

Deinitializes the internal library status.

For each call to #peak::afl::library::Init there must be a corresponding call to this function to ensure proper deinitialization of the library status.
After the library has been exited its functions (besides #peak::afl::library::Init) will not be operable until #peak::afl::library::Init has been called again.

Exceptions
peak::afl::error::Exceptionif deinitialization fails

◆ Init()

void Init ( )
inline

Init the peak_afl auto feature library.

Initializes the internal library status.

This function must be called prior to any other function call.
The function may be called multiple times from a single client process. For each call there must be a corresponding call to #peak::afl::library::Exit to ensure proper deinitialization of the library status.

Exceptions
peak::afl::error::Exceptionif initialization fails
