◆ peak_ethernet_status

enum peak_ethernet_status

peak Ethernet status

The ethernet status defines the current status of a cameras ethernet configuration.

Enumerator
PEAK_ETHERNET_STATUS_INVALID 

Invalid ethernet status.

Use this value for the initialization of variables of type peak_ethernet_status.

PEAK_ETHERNET_STATUS_OK 

The cameras ethernet status is ok.

The ethernet configuration is operable.

PEAK_ETHERNET_STATUS_NOK 

The cameras ethernet status is not ok.

The ethernet configuration is not operable.

PEAK_ETHERNET_STATUS_SUBNET_MISMATCH 

The camera persistent IP is not in the hosts subnet.

The hosts subnet is the hosts ip address AND-ed with the hosts subnet mask.
The cameras subnet is the cameras ip address AND-ed with the cameras subnet mask.
In order to have an operable ethernet connection the hosts subnet and the cameras subnet must match.

PEAK_ETHERNET_STATUS_INAPPLICABLE_IP 

The camera IP is not applicable.

The camera ip is not applicable on the network.

Make sure that the cameras persistent ip address is not used by any other device that is connected to the same network.

◆ peak_EthernetConfig_DHCP_Enable()

peak_status peak_EthernetConfig_DHCP_Enable ( peak_camera_id  cameraID,
peak_bool  enabled 
)

Enable/Disable dhcp.

Sets dhcp to enabled or disabled.

Note
If dhcp is enabled the camera will not use a possibly configured persistent ip.
If dhcp is enabled and the camera fails to acquire an address from a dhcp server in the network it may fall back to a so called link-local address. Or it may get into a configuration state with an inapplicable ip address.
Whether the camera implements the link-local fall back depends on its model and firmware implementation.
Parameters
[in]cameraIDThe camera id.
[in]enabledThe desired enabled status. PEAK_TRUE for enabled, PEAK_FALSE for disabled.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The dhcp feature is not accessible for write. Check the access status via peak_EthernetConfig_DHCP_GetAccessStatus.
PEAK_STATUS_ABORTED The could not be moved to the current subnet because no available IP was found. Therefore, dhcp could not be enabled.
PEAK_STATUS_CAMERA_NOT_FOUND There is no camera with the specified camera id.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
A change of the dhcp enabled status results in the camera re-connecting to the ethernet. For the system this is the same as if the camera is removed and re-attached.
This means that the camera needs to be newly discovered and the peak_camera_id changes after the dhcp status change.
Use peak_CameraList_Update() to initiate an update of the camera list.
Consider that the camera may need some time to re-connect to the ethernet.

◆ peak_EthernetConfig_DHCP_GetAccessStatus()

peak_access_status peak_EthernetConfig_DHCP_GetAccessStatus ( peak_camera_id  cameraID)

Query the dhcp access status.

Provides the current access status for the dhcp setting of the specified camera.

Note
The dhcp setting is supported only for cameras of the types PEAK_CAMERA_TYPE_UEYE_ETH and PEAK_CAMERA_TYPE_UEYE_PLUS_GEV.
The dhcp setting is available only for change if the camera is not opened.

This function implements the Access Status Query principle.

Parameters
[in]cameraIDThe camera id.
Returns
PEAK_ACCESS_READWRITE The dhcp setting is supported and currently available for change.
PEAK_ACCESS_READONLY The dhcp setting is supported but currently not available for change. Check for the camera to be not opened.
PEAK_ACCESS_GFA_LOCK The dhcp setting is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NOT_SUPPORTED The dhcp setting is not supported. Check for the camera type to be PEAK_CAMERA_TYPE_UEYE_PLUS_GEV.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_CAMERA_NOT_FOUND There is no camera with the specified camera id.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.
