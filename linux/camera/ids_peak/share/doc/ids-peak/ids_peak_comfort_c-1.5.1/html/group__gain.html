Detailed Description

Gain related functions and types.

This feature is applied in the camera.
There is also a host implementation. See Host Features / Gain.

◆ peak_Gain_GetAccessStatus()

peak_access_status peak_Gain_GetAccessStatus ( peak_camera_handle  hCam,
peak_gain_type  gainType,
peak_gain_channel  gainChannel 
)

Query the gain access status for the specified gain type and gain channel.

Provides the current access status of the gain feature for the specified gain type and gain channel.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
[in]gainTypeThe gain type.
[in]gainChannelThe gain channel.
Returns
PEAK_ACCESS_READWRITE The gain feature is readable and writeable.
PEAK_ACCESS_READONLY The gain feature is readable only.
PEAK_ACCESS_GFA_LOCK The gain feature is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The gain feature is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The gain feature is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_PARAMETER gainType is an invalid gain type or gainChannel is an invalid gain channel. Check peak_gain_type and peak_gain_channel.
  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_Gain_GetChannelList()

peak_status peak_Gain_GetChannelList ( peak_camera_handle  hCam,
peak_gain_type  gainType,
peak_gain_channel *  gainChannelList,
size_t *  gainChannelCount 
)

Get the list of available gain channels for the given gain type.

Queries the list of available gain channels for the specified gain type.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[in]gainTypeThe addressed gain type.
[out]gainChannelListPointer to a user allocated array buffer to receive the gain channel list. If this parameter is NULL, gainChannelCount will contain the number of gain channels.
The required size of gainChannelList in bytes is gainChannelCount x sizeof(peak_gain_channel).
[in,out]gainChannelCount
  • gainChannelList equal NULL:
    out: minimal number of gain channels gainChannelList must be large enough to hold
  • gainChannelList unequal NULL:
    in: number of gain channels gainChannelList can hold
    out: number of gain channels filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL gainChannelList is not NULL and the value of *gainChannelCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The gain feature is not accessible. Check the access status via peak_Gain_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER gainType is an invalid gain channel or gainChannelCount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
