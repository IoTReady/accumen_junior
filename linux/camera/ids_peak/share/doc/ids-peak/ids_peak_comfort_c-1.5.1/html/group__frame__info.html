Enumeration Type Documentation

◆ peak_frame_type

enum peak_frame_type

peak Frame type

The type and format of the contents of a received frame.

Enumerator
PEAK_FRAME_TYPE_INVALID 

Invalid frame type.

Use this value for the initialization of variables of type peak_frame_type.

PEAK_FRAME_TYPE_IMAGE 

Image data.

The frame contains image data, i.e. an ordered sequence of image pixels beginning at the first pixel of the image.

Check whether the frame is complete.

Incomplete Frame

This function implements the Valid Values Organization Query principle.

Returns

PEAK_FALSE The frame is incomplete or the query failed.
