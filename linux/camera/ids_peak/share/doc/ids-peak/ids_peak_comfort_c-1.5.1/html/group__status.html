Enumeration Type Documentation

◆ peak_access_status

enum peak_access_status

peak Access status codes

The access status can be queried for several features and for a camera.

See also
Access Status
Enumerator
PEAK_ACCESS_INVALID 

Invalid access status code.

Use this value for the initialization of variables of type peak_access_status.

Note
This access status code is also used by the library to indicate a failed access status query. See Access Status Query on this.
PEAK_ACCESS_NOT_SUPPORTED 

Not supported.

The feature is not supported.

Not all cameras support all features.

PEAK_ACCESS_NONE 

No access.

The feature or the camera is currently not accessible at all.

PEAK_ACCESS_GFA_LOCK 

No access.

The feature or the camera is currently not accessible due to the status of the GFA write access.
The Convenience Interface functions are not accessible if the GFA write lock is enabled.

PEAK_ACCESS_READONLY 

Read only access.

The feature is currently available for read only access.

PEAK_ACCESS_WRITEONLY 

Write only access.

The feature is currently available for write only access.

PEAK_ACCESS_READWRITE 

Read write access.

The feature or the camera is currently available for read and for write access.

◆ peak_status

enum peak_status

peak Status codes

The majority of the peak functions return a status code.

Enumerator
PEAK_STATUS_SUCCESS 

Success.

PEAK_STATUS_WARNING 

Unspecified warning.

PEAK_STATUS_VALUE_ADJUSTED 

The set value was auto-adjusted.

This status code indicates that a value which was set to a feature was automatically adjusted by the implementation or the camera.

PEAK_STATUS_WARNING_OVERFLOW 

A value was dropped.

This status code indicates that an overflow occurred. The function call nevertheless succeeded.

PEAK_STATUS_WARNING_OPERATION 

A operation failed This status code indicates that an executed operation failed. To receive the operation error code, the corresponding operation status function must be called.

PEAK_STATUS_ERROR 

Unspecified error.

This status code is typically returned on an unexpected error in the library implementation and should not show up.

PEAK_STATUS_NOT_INITIALIZED 

The library is not initialized.

Call peak_Library_Init.

PEAK_STATUS_NOT_IMPLEMENTED 

The function is not implemented.

PEAK_STATUS_ACCESS_DENIED 

The access to the requested feature was denied due to the current system status.

PEAK_STATUS_CAMERA_NOT_FOUND 

There is no such camera.

PEAK_STATUS_CAMERA_NOT_AVAILABLE 

The specified camera is not available.

PEAK_STATUS_INVALID_HANDLE 

The handle is invalid.

PEAK_STATUS_INVALID_PARAMETER 

One or more parameters are invalid.

PEAK_STATUS_OUT_OF_RANGE 

The value or configuration is out of the valid range.

PEAK_STATUS_BUFFER_TOO_SMALL 

The buffer is too small to take the complete data.

PEAK_STATUS_INVALID_CONFIGURATION 

The current configuration of an addressed module does not allow the requested operation to be executed.

PEAK_STATUS_TIMEOUT 

A timeout occurred.

PEAK_STATUS_ABORTED 

The operation was aborted.

PEAK_STATUS_NO_DATA 

The specified or requested data is not present.

PEAK_STATUS_INVALID_PEAK_INSTALLATION 

The peak installation is corrupted.

PEAK_STATUS_BUSY 

Device or resource is busy.

PEAK_STATUS_OUT_OF_MEMORY 

The system could not provide enough memory.
