enum peak_camera_memory_area

peak Camera Memory Area

A camera memory area can be used to store binary data in the cameras persistent memory.

Enumerator
PEAK_CAMERA_MEMORY_AREA_INVALID 

Invalid memory area.

Use this value for the initialization of variables of type peak_camera_memory_area.

PEAK_CAMERA_MEMORY_AREA_USER_DATA_1 

Cameras User Data 1 memory.

This memory area can be used to store binary data in the cameras persistent memory.

PEAK_CAMERA_MEMORY_AREA_USER_DATA_2 

Cameras User Data 2 memory.

This memory area can be used to store binary data in the cameras persistent memory.

◆ peak_CameraMemory_Area_GetAccessStatus()

peak_access_status peak_CameraMemory_Area_GetAccessStatus ( peak_camera_handle  hCam,
peak_camera_memory_area  cameraMemoryArea 
)

Query the camera memory area access status.

Provides the current access status for the specified camera memory area.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
[in]cameraMemoryAreaThe camera memory area.
Returns
PEAK_ACCESS_READWRITE The camera memory area can be read from and written to.
PEAK_ACCESS_READONLY The camera memory area can be read from only.
PEAK_ACCESS_WRITEONLY The camera memory area can be written to only.
PEAK_ACCESS_GFA_LOCK The camera memory area is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The camera memory area is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The camera memory area is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_PARAMETER cameraMemoryArea is an invalid parameter set. Check peak_camera_memory_area.
  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_CameraMemory_Area_GetList()

peak_status peak_CameraMemory_Area_GetList ( peak_camera_handle  hCam,
peak_camera_memory_area *  cameraMemoryAreaList,
size_t *  cameraMemoryAreaCount 
)

Get the list of currently usable camera memory areas.

Queries the list of currently selectable camera memory areas.

The list of usable camera memory areas may depend on the camera configuration and the camera status.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[out]cameraMemoryAreaListPointer to a user allocated array buffer to receive the camera memory area list. If this parameter is NULL, cameraMemoryAreaCount will contain the current number of camera memory areas.
The required size of cameraMemoryAreaList in bytes is cameraMemoryAreaCount x sizeof(peak_camera_memory_area).
[in,out]cameraMemoryAreaCount
  • cameraMemoryAreaList equal NULL:
    out: minimal number of parameter sets cameraMemoryAreaList must be large enough to hold
  • parameterSetList unequal NULL:
    in: number of camera memory areas cameraMemoryAreaList can hold
    out: number of camera memory areas filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL cameraMemoryAreaList is not NULL and the value of *cameraMemoryAreaCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The camera memory feature is not supported or the GFA write mode is enabled.
PEAK_STATUS_INVALID_PARAMETER cameraMemoryAreaCount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
Please consider that the camera memory area list might change between the size query call and the list query call.
This may be the case if the camera configuration or the camera status have changed in the time between the two function calls.
