Detailed Description

Auto brightness control related functions and types.

This feature is applied in the camera.
There is also a host implementation. See Host Features / Auto brightness control.

◆ peak_AutoBrightness_Exposure_GetAccessStatus()

peak_access_status peak_AutoBrightness_Exposure_GetAccessStatus ( peak_camera_handle  hCam)

Query the auto exposure control access status.

Provides the current access status for the auto exposure control feature.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_ACCESS_READWRITE The auto exposure control feature is readable and writeable.
PEAK_ACCESS_READONLY The auto exposure control feature is readable only.
PEAK_ACCESS_GFA_LOCK The auto exposure control feature is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The auto exposure control feature is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The auto exposure control feature is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_ACCESS_DENIED The auto brightness feature is not accessible. Check the access status via peak_AutoBrightness_GetAccessStatus.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_AutoBrightness_ROI_Offset_GetRange()

peak_status peak_AutoBrightness_ROI_Offset_GetRange ( peak_camera_handle  hCam,
peak_position *  minAutoBrightnessROIOffset,
peak_position *  maxAutoBrightnessROIOffset,
peak_position *  incAutoBrightnessROIOffset 
)

Get the current range of valid auto brightness ROI offsets.

Queries the current range of valid values for the auto brightness ROI offset.

The range of valid auto brightness ROI offset values may depend on the camera configuration and the camera status.
In special the current setting of the image ROI and the current setting of the auto brightness ROI size has an impact on the range of valid auto brightness ROI offset values.

Parameters
[in]hCamThe camera handle.
[out]minAutoBrightnessROIOffsetThe minimum auto brightness ROI offset values.
[out]maxAutoBrightnessROIOffsetThe maximum auto brightness ROI offset values.
[out]incAutoBrightnessROIOffsetThe auto brightness ROI offset values increments.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The auto brightness feature is not accessible. Check the access status via peak_AutoBrightness_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER At least one of minROIOffset, maxROIOffset, and incROIOffset is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_AutoBrightness_ROI_Set()

peak_status peak_AutoBrightness_ROI_Set ( peak_camera_handle  hCam,
peak_roi  autoBrightnessROI 
)

Set the auto brightness ROI.

Writes the desired auto brightness ROI.

Note
Setting the auto brightness ROI is only possible if PEAK_AUTO_FEATURE_ROI_MODE_MANUAL is set for the auto brightness ROI mode.
Parameters
[in]hCamThe camera handle.
[in]autoBrightnessROIThe auto brightness ROI to set.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_OUT_OF_RANGE autoBrightnessROI value is out of range. Check the range of valid values via peak_AutoBrightness_ROI_Offset_GetRange and peak_AutoBrightness_ROI_Size_GetRange.
PEAK_STATUS_ACCESS_DENIED The auto brightness feature is not available for write access or the auto brightness ROI mode is not set to PEAK_AUTO_FEATURE_ROI_MODE_MANUAL. Check the access status via peak_AutoBrightness_GetAccessStatus. Check the auto brightness ROI mode via peak_AutoBrightness_ROI_Mode_Get.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
