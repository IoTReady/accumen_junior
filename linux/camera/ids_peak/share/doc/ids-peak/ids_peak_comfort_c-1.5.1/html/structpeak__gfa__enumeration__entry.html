Detailed Description

peak GFA enumeration entry

The GFA enumeration entry describes an entry of an enumeration by its symbolic and its integer value.
The symbolic value is a text string which represents the enumeration entry.
The integer value is the integer number which represents the enumeration entry. It is also referred to as value or index of the enumeration entry.
