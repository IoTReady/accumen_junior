◆ peak_Decimation_FactorX_GetList()

peak_status peak_Decimation_FactorX_GetList ( peak_camera_handle  hCam,
uint32_t *  decimationFactorXList,
size_t *  decimationFactorXCount 
)

Get the list of currently selectable factors for the decimation in x direction.

Queries the list of currently selectable decimation factors for the x direction.

The list of selectable decimation factors may depend on the camera configuration and the camera status.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[out]decimationFactorXListPointer to a user allocated array buffer to receive the decimation factor list. If this parameter is NULL, decimationFactorXCount will contain the current number of decimation factors.
The required size of decimationFactorXList in bytes is decimationFactorXCount x sizeof(uint32_t).
[in,out]decimationFactorXCount
  • decimationFactorXList equal NULL:
    out: minimal number of decimation factors decimationFactorXList must be large enough to hold
  • decimationFactorXList unequal NULL:
    in: number of decimation factors decimationFactorXList can hold
    out: number of decimation factors filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL decimationFactorXList is not NULL and the value of *decimationFactorXCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The decimation feature is not accessible. Check the access status via peak_Decimation_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER decimationFactorXCount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
Consider that the decimation factor list might change between the size query call and the list query call.
This may be the case if the camera configuration or the camera status have changed in the time between the two function calls.
