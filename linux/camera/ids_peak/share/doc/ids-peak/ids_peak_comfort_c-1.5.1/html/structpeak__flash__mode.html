Detailed Description

peak Flash mode

A flash mode defines a combination of a flash reference and an io channel.

In order to set up a flash the desired basic configuration is specified by this struct.
Then the mode is passed to peak_Flash_Mode_Set to set the desired flash mode.
Any further configuration for the flash feature is then done via the dedicated functions.

Note
The library defines several flash mode presets which can be used to initialize a peak_flash_mode.
See
  • PEAK_FLASH_MODE_EXPOSURE_ACTIVE
  • PEAK_FLASH_MODE_ACQUISITION_ACTIVE
  • PEAK_FLASH_MODE_GLOBAL_START_WINDOW
The default flash mode is PEAK_FLASH_MODE_EXPOSURE_ACTIVE, i.e. the exposure time is reflected by the signal on the flash output channel.
The default flash mode is configured after the camera has been opened and as long as no other mode is explicitly set via peak_Flash_Mode_Set.
