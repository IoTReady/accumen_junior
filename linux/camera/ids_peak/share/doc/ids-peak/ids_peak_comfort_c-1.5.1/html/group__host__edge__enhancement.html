peak_bool  enable 
)

Enable/Disable the host edge enhancement.

Sets the edge enhancement host feature to enabled or disabled. Enabling it causes a higher CPU load.

Parameters
[in]hCamThe camera handle.
[in]enableThe desired enabled status. PEAK_TRUE for enabled, PEAK_FALSE for disabled.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.

Queries whether the edge enhancement is currently enabled or disabled.

This function implements the Enabled Status Query principle.

Parameters

Returns
PEAK_TRUE The edge enhancement feature is currently enabled.
PEAK_FALSE The edge enhancement feature is currently disabled or the query failed.
