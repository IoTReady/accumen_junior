Detailed Description

peak Camera descriptor

The camera descriptor collects information on a camera.

Use peak_Camera_GetDescriptor to query the camera descriptor for an individual camera.
Use peak_CameraList_Get to query the list of camera descriptors for all detected cameras.

Note
The user defined name within the device descriptor is only updated after a call to peak_CameraList_Update. If the device is opened, use peak_Camera_UserDefinedName_Get instead to get the current value.
For PEAK_CAMERA_TYPE_UEYE_USB and PEAK_CAMERA_TYPE_UEYE_ETH the user defined name is a numeric string.
For PEAK_CAMERA_TYPE_UEYE_PLUS_U3V non-ASCII characters may be shown as _.
