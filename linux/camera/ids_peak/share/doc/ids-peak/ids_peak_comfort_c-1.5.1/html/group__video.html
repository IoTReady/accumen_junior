Detailed Description

Functions for saving images as video file.

Typedef Documentation

◆ peak_video_handle

typedef struct peak_video* peak_video_handle

peak Video handle

The video handle repesents an opened video handle.
The video handle is provided by the peak_VideoWriter_Open function and must be released by peak_VideoWriter_Close.
The video handle is required in every function which controls a video stream.

The value for an invalid camera handle is PEAK_INVALID_HANDLE.

◆ peak_VideoWriter_Container_GetEncoderList()

peak_status peak_VideoWriter_Container_GetEncoderList ( peak_video_container  container,
peak_video_encoder *  encoderList,
size_t *  encoderCount 
)

Get the list of currently usable encoders.

Queries the list of currently selectable encoders for the given container.

This function implements the Two Staged Data Query principle.

Parameters
[in]containerContainer
[out]encoderListPointer to a user allocated array buffer to receive the encoder type list. If this parameter is NULL, encoderCount will contain the current number of encoder types.
The required size of encoderList in bytes is encoderCount x sizeof(peak_video_encoder).
[in,out]encoderCount
  • encoderList equal NULL:
    out: minimal number of encoder types encoderList must be large enough to hold
  • encoderList unequal NULL:
    in: number of encoder types encoderList can hold
    out: number of encoder types filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL encoderList is not NULL and the value of *encoderCount is too small to receive the expected amount of data.
PEAK_STATUS_INVALID_PARAMETER encoder is an invalid container or encoderCount is an invalid pointer.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
