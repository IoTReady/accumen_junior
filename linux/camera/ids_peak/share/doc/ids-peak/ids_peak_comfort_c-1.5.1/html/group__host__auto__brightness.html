◆ peak_IPL_AutoBrightness_ROI_Set()

peak_status peak_IPL_AutoBrightness_ROI_Set ( peak_camera_handle  hCam,
peak_roi  autoBrightnessROI 
)

Set the host auto brightness ROI.

Writes the desired host auto brightness ROI.

Note
Setting the auto brightness ROI is only possible if PEAK_AUTO_FEATURE_ROI_MODE_MANUAL is set for the auto brightness ROI mode.
The range of valid ROIs for the auto brightness feature is the range of valid image ROIs. It can be queried via peak_IPL_AutoBrightness_ROI_Offset_GetRange and peak_IPL_AutoBrightness_ROI_Size_GetRange.
Parameters
[in]hCamThe camera handle.
[in]autoBrightnessROIThe auto brightness ROI to set.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_OUT_OF_RANGE autoBrightnessROI value is out of range. Check the range of valid values via peak_IPL_AutoBrightness_ROI_Offset_GetRange and peak_IPL_AutoBrightness_ROI_Size_GetRange.
PEAK_STATUS_ACCESS_DENIED The auto brightness ROI mode is not set to PEAK_AUTO_FEATURE_ROI_MODE_MANUAL. Check the auto brightness ROI mode via peak_IPL_AutoBrightness_ROI_Mode_Get.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
