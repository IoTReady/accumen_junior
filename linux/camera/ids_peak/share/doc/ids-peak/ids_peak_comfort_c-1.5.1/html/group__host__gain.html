Get the host gain value for the specified gain channel.

Reads the current host gain value for the specified gain channel.

Parameters
[in]hCamThe camera handle.
[in]gainChannelThe addressed gain channel.
[out]gainThe gain value.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_INVALID_PARAMETER gainChannel is an invalid gain channel or gain is an invalid pointer. Check peak_gain_channel.

PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
