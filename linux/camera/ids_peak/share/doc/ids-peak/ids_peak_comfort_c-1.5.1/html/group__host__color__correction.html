Detailed Description

Color correction related functions and types.

This feature is applied in the host.
There is also a camera implementation. See Color correction.

Get the enabled status of the host color correction.

Queries whether the host color correction is currently enabled or disabled.

This function implements the Enabled Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_TRUE The color correction feature is currently enabled.
PEAK_FALSE The color correction feature is currently disabled or the query failed.
