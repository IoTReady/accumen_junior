◆ peak_ROI_GetAccessStatus()

peak_access_status peak_ROI_GetAccessStatus ( peak_camera_handle  hCam)

Query the ROI access status.

Provides the current access status for the ROI feature.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_ACCESS_READWRITE The ROI feature is readable and writeable.
PEAK_ACCESS_READONLY The ROI feature is readable only.
PEAK_ACCESS_GFA_LOCK The ROI feature is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The ROI feature is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The ROI feature is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

Get the current range of valid ROI offsets.

Queries the current range of valid values for the ROI offset.

The range of valid ROI offset values may depend on the camera configuration and the camera status.
In special the current setting of the ROI size has an impact on the range of valid ROI offset values.

Parameters
[in]hCamThe camera handle.
[out]minROIOffsetThe minimum ROI offset values.
[out]maxROIOffsetThe maximum ROI offset values.
[out]incROIOffsetThe ROI offset values increments.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The ROI feature is not accessible. Check the access status via peak_ROI_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER At least one of minROIOffset, maxROIOffset, and incROIOffset is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
