◆ peak_ExposureTime_GetAccessStatus()

peak_access_status peak_ExposureTime_GetAccessStatus ( peak_camera_handle  hCam)

Query the exposure time access status.

Provides the current access status for the exposure time control.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_ACCESS_READWRITE The exposure time control is readable and writeable.
PEAK_ACCESS_READONLY The exposure time control is readable only.
PEAK_ACCESS_GFA_LOCK The exposure time control is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The exposure time control is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The exposure time control is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

Get the current range of valid exposure time values.

Queries the current range of valid values for the exposure time control.

The range of valid exposure time values may depend on the camera configuration and the camera status.

Parameters
[in]hCamThe camera handle.
[out]minExposureTime_usThe minimum exposure time in microseconds.
[out]maxExposureTime_usThe maximum exposure time in microseconds.
[out]incExposureTime_usThe exposure time increment in microseconds.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The exposure time control is not accessible. Check the access status via peak_ExposureTime_GetAccessStatus.

PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
