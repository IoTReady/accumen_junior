peak_status peak_GFA_Integer_GetList ( peak_camera_handle  hCam,
peak_gfa_module  module,
const char *  integerFeatureName,

)

Get the list of currently selectable values for the integer feature with the given name.

Queries the list of currently selectable values for the integer feature with the given name.

The list of selectable integer values may depend on the camera configuration and the camera status.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[in]moduleThe addressed module.
[in]integerFeatureNameThe name of the integer feature as a zero-terminated string.
[out]integerListPointer to a user allocated array buffer to receive the integer value list. If this parameter is NULL, integerCount will contain the current number of integer values.
The required size of integerList in bytes is integerCount x sizeof(int64_t).
[in,out]integerCount
  • integerList equal NULL:
    out: minimal number of integer values integerList must be large enough to hold
  • integerList unequal NULL:
    in: number of integer values integerList can hold
    out: number of integer values filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_NO_DATA The valid values for the integer feature are organized as range rather than as a list.
Use peak_GFA_Integer_GetRange to query the valid integer values.
PEAK_STATUS_BUFFER_TOO_SMALL integerList is not NULL and the value of *integerCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The integer feature is not accessible. Check the access status via peak_GFA_Feature_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER module is an invalid GFA module. or integerFeatureName and/or integerCount are invalid pointers or integerFeatureName is not zero-terminated. Check peak_gfa_module.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
Consider that the integer value list might change between the size query call and the list query call.
This may be the case if the camera configuration or the camera status have changed in the time between the two function calls.

peak_gfa_module  module,
const char *  integerFeatureName 
)

Check whether the valid values for the integer feature with the given name are organized as a range.

Checks whether the valid values for the integer feature with the given name are organized as a range or as a list.

This function implements the Valid Values Organization Query principle.

Parameters
[in]hCamThe camera handle.
[in]moduleThe addressed module.
[in]integerFeatureNameThe name of the integer feature as a zero-terminated string.
Returns
PEAK_TRUE The valid values are organized as a range. Use peak_GFA_Integer_GetRange.
PEAK_FALSE The valid values are organized as a list. Use peak_GFA_Integer_GetList.
