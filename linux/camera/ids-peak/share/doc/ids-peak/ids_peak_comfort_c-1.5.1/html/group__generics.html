enum peak_auto_feature_mode

peak Auto feature mode

The auto feature mode defines the mode of operation for auto features.

Enumerator
PEAK_AUTO_FEATURE_MODE_INVALID 

Invalid auto feature mode.

Use this value for the initialization of variables of type peak_auto_feature_mode.

PEAK_AUTO_FEATURE_MODE_OFF 

Auto feature Off.

The auto feature is off.

This is the default mode for all auto features.

PEAK_AUTO_FEATURE_MODE_ONCE 

Auto feature mode Once.

The auto feature is executed once.

PEAK_AUTO_FEATURE_MODE_CONTINUOUS 

Auto feature mode Continuous.

The auto feature is executed continuously.

enum peak_auto_feature_roi_mode

peak Auto feature ROI mode

The auto feature ROI mode defines the mode of operation for auto feature ROIs.

Enumerator
PEAK_AUTO_FEATURE_ROI_MODE_INVALID 

Invalid auto feature ROI mode.

Use this value for the initialization of variables of type peak_auto_feature_roi_mode.

PEAK_AUTO_FEATURE_ROI_MODE_FULL_IMAGE 

Auto feature ROI mode Full Image.

The auto feature ROI automatically follows the image ROI.
I.e. the auto feature always covers the whole image.

This is the default mode for all auto features.

PEAK_AUTO_FEATURE_ROI_MODE_MANUAL 

Auto feature ROI mode Manual.

The auto feature ROI can be set manually inside of the bounds of the image ROI.
If the image ROI size is reduced the auto feature ROI may be automatically clipped. If the auto feature ROI is out of the image ROI bounds after a change of the image ROI, the position of the auto feature ROI is adjusted and its size is set to minimum.
