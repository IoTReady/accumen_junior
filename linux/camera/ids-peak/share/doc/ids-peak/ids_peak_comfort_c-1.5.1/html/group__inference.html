◆ peak_Inference_Result_Classification_GetList()

peak_status peak_Inference_Result_Classification_GetList ( peak_inference_result_handle  hInferenceHandle,
peak_inference_result_classification *  resultList,
size_t *  resultCount 
)

Get the result entries from a given classification inference result handle.

Provides the current list of result entries for classification

This function implements the Two Staged Data Query principle.

Parameters
[in]hInferenceHandleHandle to an inference result.
[out]resultListPointer to a user allocated array buffer to receive the result entry list. If this parameter is NULL, resultCount will contain the current number of result entries.
The required size for resultList in bytes is resultCount x sizeof(peak_inference_result_classification)
[in,out]resultCount
  • resultList equal NULL:
    out: minimal number of result entries resultList must be large enough to hold
  • resultList unequal NULL:
    in: number of result entries resultList can hold
    out: number of result entries filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_INVALID_PARAMETER resultCount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE resultHandle is an invalid handle.
PEAK_STATUS_BUFFER_TOO_SMALL resultList is not NULL and the value of *resultCount is too small to receive the expected amount of data.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
