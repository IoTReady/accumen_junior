Detailed Description

peak Frame info

The frame info collects information on an acquired frame.

Field Documentation

◆ bytesExpected

size_t bytesExpected

The number of bytes expected for a complete image.

This value may be smaller than buffer.memorySize

  • if the buffer was announced via peak_Acquisition_Buffer_Announce and is larger than required.

◆ bytesWritten

size_t bytesWritten

The number of bytes actually written to the buffer.

This value may be smaller than buffer.memorySize

  • if the frame is incomplete (see isIncomplete), or
  • if the buffer was announced via peak_Acquisition_Buffer_Announce and is larger than required. See bytesExpected.

◆ frameID

uint64_t frameID

The frame id.

The frame id is incremented with every acquired frame. It is reset to 0 on peak_Acquisition_Start.

The frame id may jump forwards from one delivered frame to another. A jump in the frame id indicates a Dropped Frame.
