◆ peak_Flash_EndDelay_GetAccessStatus()

peak_access_status peak_Flash_EndDelay_GetAccessStatus ( peak_camera_handle  hCam)

Query the flash end delay property access status.

Provides the current access status for the flash end delay property for the configured flash mode.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_ACCESS_READWRITE The flash end delay property is readable and writeable.
PEAK_ACCESS_READONLY The flash end delay property is readable only.
PEAK_ACCESS_GFA_LOCK The flash end delay property is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The flash end delay property is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The flash end delay property is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_ACCESS_DENIED The flash feature is not accessible. Check the access status of the flash feature via peak_Flash_GetAccessStatus.
  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

Get the current range of valid flash end delay values.

Queries the current range of valid values for the flash end delay for the configured flash mode.

The range of valid flash end delay values may depend on the flash mode, the camera configuration, and the camera status.

Parameters
[in]hCamThe camera handle.
[out]minFlashEndDelay_usThe minimum flash end delay value in microseconds.
[out]maxFlashEndDelay_usThe maximum flash end delay value in microseconds.
[out]incFlashEndDelay_usThe flash end delay value increment in microseconds.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The flash end delay property is not accessible. Check the access status of the flash end delay via peak_Flash_EndDelay_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER At least one of minFlashEndDelay_us, maxFlashEndDelay_us, and incFlashEndDelay_us is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
