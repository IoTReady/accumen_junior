◆ peak_IPL_AutoWhiteBalance_ROI_Set()

peak_status peak_IPL_AutoWhiteBalance_ROI_Set ( peak_camera_handle  hCam,
peak_roi  autoWhiteBalanceROI 
)

Set the host auto white balance ROI.

Writes the desired host auto white balance ROI.

Note
Setting the auto white balance ROI is only possible if PEAK_AUTO_FEATURE_ROI_MODE_MANUAL is set for the auto white balance ROI mode.
The range of valid ROIs for the auto white balance feature is the range of valid image ROIs. It can be queried via peak_IPL_AutoWhiteBalance_ROI_Offset_GetRange and peak_IPL_AutoWhiteBalance_ROI_Size_GetRange.
Parameters
[in]hCamThe camera handle.
[in]autoWhiteBalanceROIThe auto white balance ROI to set.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_OUT_OF_RANGE autoWhiteBalanceROI value is out of range. Check the range of valid values via peak_IPL_AutoWhiteBalance_ROI_Offset_GetRange and peak_IPL_AutoWhiteBalance_ROI_Size_GetRange.
PEAK_STATUS_ACCESS_DENIED The auto white balance ROI mode is not set to PEAK_AUTO_FEATURE_ROI_MODE_MANUAL. Check the auto white balance ROI mode via peak_IPL_AutoWhiteBalance_ROI_Mode_Get.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
