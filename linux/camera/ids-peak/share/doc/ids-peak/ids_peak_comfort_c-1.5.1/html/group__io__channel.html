◆ peak_io_channel

enum peak_io_channel

peak IO channel

The io channel specifies the channel at which a certain signal or event is received or transmitted. It is used for several features like trigger and flash.

Not all channels are applicable for all features.

Enumerator
PEAK_IO_CHANNEL_INVALID 

Invalid io channel.

Use this value for the initialization of variables of type peak_io_channel;.

PEAK_IO_CHANNEL_NONE 

None.

No channel configured.

PEAK_IO_CHANNEL_SOFTWARE 

Software.

The signal or event is initiated by a software function.

PEAK_IO_CHANNEL_TRIGGER_INPUT 

Trigger input pin.

The signal or event is initiated by an electrical signal at the cameras trigger connector.

PEAK_IO_CHANNEL_FLASH_OUTPUT 

Flash output pin.

The signal or event is initiated by an electrical signal at the cameras flash connector.

PEAK_IO_CHANNEL_GPIO_1 

GPIO 1 pin.

The signal or event is initiated by an electrical signal at the cameras GPIO 1 connector.

PEAK_IO_CHANNEL_GPIO_2 

GPIO 2 pin.

The signal or event is initiated by an electrical signal at the cameras GPIO 2 connector.

◆ peak_IOChannel_GetAccessStatus()

peak_access_status peak_IOChannel_GetAccessStatus ( peak_camera_handle  hCam,
peak_io_channel  ioChannel 
)

Query the io channel access status.

Provides the current access status for the specified io channel.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
[in]ioChannelThe io channel.
Returns
PEAK_ACCESS_READWRITE The io channel is supported and currently available for use by trigger/flash.
PEAK_ACCESS_READONLY The io channel is supported but currently used in an enabled trigger/flash configuration.
PEAK_ACCESS_GFA_LOCK The io channel is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NOT_SUPPORTED The io channel is not supported. Check the list of supported io channels via peak_IOChannel_GetList.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_ACCESS_DENIED The io channel feature is not supported.
  • PEAK_STATUS_INVALID_PARAMETER ioChannel is an invalid io channel. Check peak_io_channel.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_IOChannel_GetList()

peak_status peak_IOChannel_GetList ( peak_camera_handle  hCam,
peak_io_channel *  ioChannelList,
size_t *  ioChannelCount 
)

Get the list of supported io channels.

Queries the list of supported io channels.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[out]ioChannelListPointer to a user allocated array buffer to receive the io channel list. If this parameter is NULL, ioChannelCount will contain the current number of io channels.
The required size of ioChannelList in bytes is ioChannelCount x sizeof(peak_io_channel).
[in,out]ioChannelCount
  • ioChannelList equal NULL:
    out: minimal number of io channels ioChannelList must be large enough to hold
  • ioChannelList unequal NULL:
    in: number of io channels ioChannelList can hold
    out: number of io channels filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL ioChannelList is not NULL and the value of *ioChannelCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The io channel feature is not supported or the GFA write access is enabled.
PEAK_STATUS_INVALID_PARAMETER ioChannelCount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
