Get the camera handle for a message.

May be PEAK_INVALID_HANDLE if no camera is associated with the message.

Parameters
[in]hMessageHandle for a message instance.
[out]hCamThe camera handle of the origin camera or PEAK_INVALID_HANDLE if origin is not the camera e.g. the camera manager.
Returns

PEAK_STATUS_INVALID_HANDLE hMessage is an invalid handle.
PEAK_STATUS_INVALID_PARAMETER hCam is an invalid pointer.

PEAK_STATUS_ERROR An unexpected internal error occurred.
