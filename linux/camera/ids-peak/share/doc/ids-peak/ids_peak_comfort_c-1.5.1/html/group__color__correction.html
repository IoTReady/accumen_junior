Detailed Description

Color correction related functions and types.

This feature is applied in the camera.
There is also a host implementation. See Host Features / Color correction.

Enumeration Type Documentation

◆ peak_color_correction_mode

enum peak_color_correction_mode

peak Color correction mode

The color correction mode defines the matrix of factors that the color correction feature applies to the image data.

Enumerator
PEAK_COLOR_CORRECTION_MODE_INVALID 

Invalid color correction.

Use this value for the initialization of variables of type peak_color_correction.

PEAK_COLOR_CORRECTION_MODE_HQ 

HQ.

The color correction is done with the high quality matrix preset.

PEAK_COLOR_CORRECTION_MODE_USER_1 

User 1.

The color correction is done with the user-defined matrix.

By default the user-defined matrix is set to the identity matrix.
It can be changed via peak_ColorCorrection_Matrix_Set.

◆ peak_ColorCorrection_GetAccessStatus()

peak_access_status peak_ColorCorrection_GetAccessStatus ( peak_camera_handle  hCam)

Query the color correction access status.

Provides the current access status for the color correction feature.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_ACCESS_READWRITE The color correction feature is readable and writeable.
PEAK_ACCESS_READONLY The color correction feature is readable only.
PEAK_ACCESS_GFA_LOCK The color correction feature is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The color correction feature is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The color correction feature is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_ColorCorrection_Mode_GetList()

peak_status peak_ColorCorrection_Mode_GetList ( peak_camera_handle  hCam,
peak_color_correction_mode *  colorCorrectionModeList,
size_t *  colorCorrectionModeCount 
)

Get the list of currently selectable color correction modes.

Queries the list of currently selectable color correction modes.

The list of selectable color correction modes may depend on the camera configuration and the camera status.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[out]colorCorrectionModeListPointer to a user allocated array buffer to receive the color correction mode list.
If this parameter is NULL, colorCorrectionModeCount will contain the current number of color correction modes.
The required size of colorCorrectionModeList in bytes is colorCorrectionModeCount x sizeof(peak_color_correction_mode).
[in,out]colorCorrectionModeCount
  • colorCorrectionModeList equal NULL:
    out: minimal number of color correction modes colorCorrectionModeList must be large enough to hold
  • colorCorrectionModeList unequal NULL:
    in: number of color corrections modes colorCorrectionModeList can hold
    out: number of color correction modes filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL colorCorrectionModeList is not NULL and the value of *colorCorrectionModeCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The color correction feature is not accessible. Check the access status via peak_ColorCorrection_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER colorCorrectionModeCount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
Please consider that the color correction mode list might change between the size query call and the list query call.
This may be the case if the camera configuration or the camera status have changed in the time between the two function calls.
