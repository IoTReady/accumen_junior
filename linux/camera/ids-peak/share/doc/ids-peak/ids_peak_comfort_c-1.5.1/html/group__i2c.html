◆ peak_I2C_AckPolling_Enable()

peak_status peak_I2C_AckPolling_Enable ( peak_i2c_handle  hI2C,
peak_bool  enabled 
)

Enable/Disable the i2c ack polling.

Sets the i2c ack polling to enabled or disabled. When i2c ack polling is enabled, the camera waits for an acknowledge after an I2C write operation. If this acknowledge is not received within the ack polling timeout, the i2c operation status changes to a timeout error.

See also
peak_I2C_AckPolling_Timeout_Set
peak_I2C_OperationStatus_Get
Parameters
[in]hI2CThe i2c handle.
[in]enabledThe desired enabled status. PEAK_TRUE for enabled, PEAK_FALSE for disabled.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The i2c feature is not accessible for write. Check the access status of the i2c feature via peak_I2C_GetAccessStatus or check the last error for more information via peak_Library_GetLastError.
PEAK_STATUS_INVALID_HANDLE hI2C is an invalid i2c handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_I2C_GetAccessStatus()

peak_access_status peak_I2C_GetAccessStatus ( peak_i2c_handle  hI2C)

Query the i2c access status.

Provides the current access status for the specified i2c handle.

This function implements the Access Status Query principle.

Parameters
[in]hI2CThe i2c handle.
Returns
PEAK_ACCESS_READWRITE The i2c can be read from and written to.
PEAK_ACCESS_READONLY The i2c can be read from only.
PEAK_ACCESS_WRITEONLY The i2c can be written to only.
PEAK_ACCESS_GFA_LOCK The i2c is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The i2c is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The i2c is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hI2C is an invalid i2c handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_I2C_Mode_GetList()

peak_status peak_I2C_Mode_GetList ( peak_i2c_handle  hI2C,
peak_i2c_mode *  modeList,
size_t *  modeListSize 
)

Get the list of currently selectable i2c modes.

Queries the list of currently selectable i2c modes.

The list of selectable i2c modes may depend on the trigger mode, the camera configuration, and the camera status.

This function implements the Two Staged Data Query principle.

Parameters
[in]hI2CThe i2c handle.
[out]modeListPointer to a user allocated array buffer to receive the i2c mode list. If this parameter is NULL, modeListSize will contain the current number of i2c modes.
The required size of modeList in bytes is modeListSize x sizeof(peak_i2c_mode).
[in,out]modeListSize
  • modeList equal NULL:
    out: minimal number of i2c modes modeList must be large enough to hold
  • modeList unequal NULL:
    in: number of i2c modes modeList can hold
    out: number of i2c modes filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_BUFFER_TOO_SMALL modeList is not NULL and the value of *modeListSize is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The i2c modes property is not accessible. Check the access status of the i2c feature via peak_I2C_GetAccessStatus or check the last error for more information via peak_Library_GetLastError.
PEAK_STATUS_INVALID_PARAMETER modeListSize is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hI2C is an invalid i2c handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
Consider that the i2c mode list might change between the size query call and the list query call.
This may be the case if the i2c mode, the camera configuration, or the camera status have changed in the time between the two function calls.
