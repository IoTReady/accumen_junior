peak_status peak_GFA_Float_GetList ( peak_camera_handle  hCam,
peak_gfa_module  module,
const char *  floatFeatureName,

)

Get the list of currently selectable values for the float feature with the given name.

Queries the list of currently selectable values for the float feature with the given name.

The list of selectable float values may depend on the camera configuration and the camera status.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[in]moduleThe addressed module.
[in]floatFeatureNameThe name of the float feature as a zero-terminated string.
[out]floatListPointer to a user allocated array buffer to receive the float value list. If this parameter is NULL, floatCount will contain the current number of float values.
The required size of floatList in bytes is floatCount x sizeof(double).
[in,out]floatCount
  • floatList equal NULL:
    out: minimal number of float values floatList must be large enough to hold
  • floatList unequal NULL:
    in: number of float values floatList can hold
    out: number of float values filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_NO_DATA The valid values for the float feature are organized as range rather than as a list.
Use peak_GFA_Float_GetRange to query the valid float values.
PEAK_STATUS_BUFFER_TOO_SMALL floatList is not NULL and the value of *floatCount is too small to receive the expected amount of data.
PEAK_STATUS_ACCESS_DENIED The float feature is not accessible. Check the access status via peak_GFA_Feature_GetAccessStatus.
PEAK_STATUS_INVALID_PARAMETER module is an invalid GFA module or floatFeatureName and/or floatCount are invalid pointers or floatFeatureName is not zero-terminated. Check peak_gfa_module.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
Note
Consider that the float value list might change between the size query call and the list query call.
This may be the case if the camera configuration or the camera status have changed in the time between the two function calls.

Check whether the valid values for the float feature with the given name are organized as a range.

Checks whether the valid values for the float feature with the given name are organized as a range or as a list.

This function implements the Valid Values Organization Query principle.

Parameters
[in]hCamThe camera handle.
[in]moduleThe addressed module.
[in]floatFeatureNameThe name of the float feature as a zero-terminated string.
Returns
PEAK_TRUE The valid values are organized as a range. Use peak_GFA_Float_GetRange.
PEAK_FALSE The valid values are organized as a list. Use peak_GFA_Float_GetList.
