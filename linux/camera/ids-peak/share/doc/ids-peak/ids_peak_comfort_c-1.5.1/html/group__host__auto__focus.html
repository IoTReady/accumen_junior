Detailed Description

Auto focus related functions and types.

After each configuration change, the autofocus must be retriggered. This is necessary for both "Once" and "Continuous" mode.

◆ peak_auto_focus_search_algorithm

enum peak_auto_focus_search_algorithm

The peak focus search algorithm.

Enumerator
PEAK_AUTO_FOCUS_SEARCH_ALGORITHM_INVALID 

Invalid search algorithm type.

Use this value for the initialization of variables of type peak_search_algorithm.

PEAK_AUTO_FOCUS_SEARCH_ALGORITHM_GOLDEN_RATIO 

Search algorithm Golden Ratio.

Searches and refines the maximum search by subdividing the search area by the golden ratio.

PEAK_AUTO_FOCUS_SEARCH_ALGORITHM_HILL_CLIMBING 

Search algorithm Hill Climbing.

Detects a maximum when the sharpness values no longer increase and aborts searching in the following focus area with the first decrease in sharpness value.

PEAK_AUTO_FOCUS_SEARCH_ALGORITHM_FULL_SCAN 

Search algorithm Full Scan.

Calculates the sharpness values in the entire search range in a single pass without a search strategy.

PEAK_AUTO_FOCUS_SEARCH_ALGORITHM_GLOBAL_SEARCH 

Search algorithm Full Scan.

Starts the coarse search in the far range with a large search interval.

◆ peak_IPL_AutoFocus_GetAccessStatus()

peak_access_status peak_IPL_AutoFocus_GetAccessStatus ( peak_camera_handle  hCam)

Query the auto focus access status.

Provides the current access status for the auto focus feature.

This function implements the Access Status Query principle.

Parameters
[in]hCamThe camera handle.
Returns
PEAK_ACCESS_READWRITE The auto focus feature is readable and writeable.
PEAK_ACCESS_READONLY The auto focus feature is readable only.
PEAK_ACCESS_GFA_LOCK The auto focus feature is not accessible because the GFA write access is enabled.
PEAK_ACCESS_NONE The auto focus feature is not accessible.
PEAK_ACCESS_NOT_SUPPORTED The auto focus feature is not supported.
PEAK_ACCESS_INVALID The function failed. Call peak_Library_GetLastError to get the error code and description.

If the function indicates an error by returning PEAK_ACCESS_INVALID, these are the possible errors:

  • PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
  • PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
  • PEAK_STATUS_ERROR An unexpected internal error occurred.

◆ peak_IPL_AutoFocus_ROI_Get()

peak_status peak_IPL_AutoFocus_ROI_Get ( peak_camera_handle  hCam,
peak_focus_roi *  autoFocusROIList,
size_t *  autoFocusROICount 
)

Get the list of host auto focus regions of interest.

Reads the current host auto focus ROI list.

This function implements the Two Staged Data Query principle.

Parameters
[in]hCamThe camera handle.
[out]autoFocusROIListPointer to a user allocated array buffer to receive the focus ROI list. If this parameter is NULL, autoFocusROICount will contain the current number of ROIs.
The required size of autoFocusROIList in bytes is autoFocusROICount x sizeof(peak_focus_roi).
[in,out]autoFocusROICount
  • autoFocusROIList equal NULL:
    out: minimal number of focus ROIs autoFocusROIList must be large enough to hold
  • autoFocusROIList unequal NULL:
    in: number of ROIs autoFocusROIList can hold
    out: number of ROIs filled by the function
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_ACCESS_DENIED The auto focus feature is not accessible. Check the access status via peak_IPL_AutoFocus_GetAccessStatus.
PEAK_STATUS_BUFFER_TOO_SMALL autoFocusROIList is not NULL and the value of *autoFocusROICount is too small to receive the expected amount of data.
PEAK_STATUS_INVALID_PARAMETER autoFocusROICount is an invalid pointer.
PEAK_STATUS_INVALID_HANDLE hCam is an invalid camera handle.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
