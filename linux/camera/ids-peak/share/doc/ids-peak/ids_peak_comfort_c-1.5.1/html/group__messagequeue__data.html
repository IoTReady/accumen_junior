◆ peak_Message_Data_Type_Get()

peak_status peak_Message_Data_Type_Get ( peak_message_handle  hMessage,
peak_message_data_type *  messageType 
)

Get the message data type.

This is important to get the associated message data for a message. The following list describes which function should be called for each type to get the data. For the device events a call to peak_Message_Data_RemoteDevice_Get is always valid.

  • PEAK_MESSAGE_DATA_TYPE_NO_DATA: No data is associated with the message.
  • PEAK_MESSAGE_DATA_TYPE_REMOTE_DEVICE: Call peak_Message_Data_RemoteDevice_Get.
  • PEAK_MESSAGE_DATA_TYPE_REMOTE_DEVICE_ERROR: Call peak_Message_Data_RemoteDeviceError_Get.
  • PEAK_MESSAGE_DATA_TYPE_REMOTE_DEVICE_DROPPED: Call peak_Message_Data_RemoteDeviceDropped_Get.
  • PEAK_MESSAGE_DATA_TYPE_REMOTE_DEVICE_FRAME: Call peak_Message_Data_RemoteDeviceFrame_Get.
  • PEAK_MESSAGE_DATA_TYPE_REMOTE_DEVICE_TEMPERATURE: Call peak_Message_Data_RemoteDeviceTemperature_Get.
  • PEAK_MESSAGE_DATA_TYPE_AUTOFOCUS_DATA: Call peak_Message_Data_AutoFocusData_Get.
Parameters
[in]hMessageHandle for a message instance.
[out]messageTypeThe message data type.
Returns
PEAK_STATUS_SUCCESS Operation was successful; no error occurred.
PEAK_STATUS_INVALID_HANDLE hMessage is an invalid handle.
PEAK_STATUS_INVALID_PARAMETER messageType is an invalid pointer.
PEAK_STATUS_NOT_INITIALIZED The library is not initialized.
PEAK_STATUS_ERROR An unexpected internal error occurred.
